Topic 4

List Tags

HTML lists are used to display collections of items. There are three types: unordered lists (bullets), ordered lists (numbers), and description lists (terms with definitions).

The Three Types of HTML Lists

TypeTagDescription
Unordered List<ul>Items displayed with bullet points — no particular order
Ordered List<ol>Items displayed with numbers or letters — in a specific order
Description List<dl>Terms and their descriptions — like a glossary

1. Unordered List <ul>

An unordered list displays items with bullet points. Each item is wrapped in <li> tags. The type attribute changes the bullet style.

AttributeBullet Style
<ul type="disc">● Filled circle (default)
<ul type="circle">○ Hollow circle
<ul type="square">■ Square
HTML — Unordered List
<ul type="disc">
  <li>Beetroot</li>
  <li>Ginger</li>
  <li>Potato</li>
  <li>Radish</li>
</ul>
Output
  • Beetroot
  • Ginger
  • Potato
  • Radish

2. Ordered List <ol>

An ordered list numbers items sequentially. The type attribute changes the numbering style. The start attribute sets the starting number.

AttributeNumbering Style
<ol type="1">1, 2, 3, 4 (default numerals)
<ol type="A">A, B, C, D (uppercase letters)
<ol type="a">a, b, c, d (lowercase letters)
<ol type="I">I, II, III, IV (uppercase Roman)
<ol type="i">i, ii, iii, iv (lowercase Roman)
HTML — Ordered List (Uppercase Letters, Start at D)
<ol type="A" start="4">
  <li>HTML Basics</li>
  <li>CSS Styling</li>
  <li>JavaScript</li>
</ol>
Output
  1. HTML Basics
  2. CSS Styling
  3. JavaScript
HTML — Ordered List (Roman Numerals)
<ol type="I">
  <li>Introduction</li>
  <li>Main Content</li>
  <li>Conclusion</li>
</ol>
Output
  1. Introduction
  2. Main Content
  3. Conclusion

3. Description List <dl>

A description list is used to display terms (<dt>) and their definitions (<dd>), like a glossary or dictionary.

TagMeaning
<dl>Description List — wraps the whole list
<dt>Description Term — the word or phrase being defined
<dd>Description Definition — the explanation of the term
HTML — Description List
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language — used to structure web pages</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets — used to style web pages</dd>

  <dt>JavaScript</dt>
  <dd>A programming language used to add interactivity</dd>
</dl>
Output
HTML
HyperText Markup Language — used to structure web pages
CSS
Cascading Style Sheets — used to style web pages
JavaScript
A programming language used to add interactivity

Nested Lists

You can nest a list inside another list item to create a hierarchy:

HTML — Nested List
<ul>
  <li>Front-End
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>Back-End
    <ol>
      <li>Python</li>
      <li>Node.js</li>
    </ol>
  </li>
</ul>
Output
  • Front-End
    • HTML
    • CSS
    • JavaScript
  • Back-End
    1. Python
    2. Node.js

Quick Reference Summary

<ul> — Unordered list <ol> — Ordered list <li> — List item <dl> — Description list <dt> — Term <dd> — Definition
