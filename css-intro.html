Introduction to CSS

CSS (Cascading Style Sheets) is used to control the presentation and layout of HTML elements — colours, fonts, spacing, positioning, animations, and responsive design.

What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML elements are displayed on screen.

The word cascading means that a style applied to a parent element will also apply to all its children. For example, setting color: blue on the <body> will make all paragraphs and headings inside it blue too.

Why Use CSS?

Without CSS, all web pages would look like plain, unstyled text documents. CSS allows you to:

  • Control colours, fonts, and text sizes
  • Control the layout and positioning of elements
  • Add backgrounds, borders, and spacing
  • Create responsive designs for mobile and desktop
  • Add animations and transitions
  • Control multiple pages from one file (external CSS)

CSS Syntax

A CSS rule is made of three parts: a selector, a property, and a value.

CSS — Basic Syntax
selector {
  property: value;
}

/* Example: */
body {
  background-color: powderblue;
  color: #333333;
  font-size: 16px;
}
PartDescriptionExample
SelectorThe HTML element to stylebody, h1, p, .class, #id
PropertyWhat aspect to changecolor, font-size, background-color
ValueThe setting for that propertyred, 16px, #ffffff

Three Ways to Add CSS

1. Inline CSS

Applied directly to a single HTML element using the style attribute. Used for quick, one-off styles.

HTML — Inline CSS
<h1 style="color: red; font-size: 32px;">Red Heading</h1>
<p style="color: blue; font-family: Arial;">Blue paragraph</p>
Output

Red Heading

Blue paragraph

2. Internal (Embedded) CSS

Placed inside a <style> block in the <head> section. Only applies to that one HTML page.

HTML — Internal CSS
<head>
  <style>
    h1 { color: green; text-decoration: underline; }
    p  { font-size: 18px; font-family: sans-serif; }
  </style>
</head>

3. External CSS (Best Practice ⭐)

A separate .css file linked to the HTML page. This is the recommended approach — one file can style an entire website.

HTML — Linking External CSS
<head>
  <title>My Website</title>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>
CSS — style.css
body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

h1 {
  color: navy;
  font-size: 36px;
}

p {
  font-size: 16px;
  line-height: 1.6;
}

The CSS Box Model

Every HTML element is a rectangular box. The CSS box model describes the layers around it:

MARGIN
BORDER
PADDING
CONTENT
LayerDescription
ContentThe actual text or image — controlled by width and height
PaddingSpace INSIDE the border, between content and border (transparent)
BorderA line that goes around the padding and content
MarginSpace OUTSIDE the border, between this element and others (transparent)
CSS — Box Model Example
div {
  width: 300px;
  padding: 20px;
  border: 3px solid #006644;
  margin: 15px;
  border-radius: 8px;
}

Media Queries — Responsive Design

Media queries apply different styles based on the screen size, making websites look good on phones, tablets, and desktops.

CSS — Media Query Example
/* Desktop (large screens above 1024px) */
@media (min-width: 1025px) {
  body { font-size: 18px; }
}

/* Tablet (768px to 1024px) */
@media (min-width: 768px) and (max-width: 1024px) {
  body { font-size: 16px; }
}

/* Smartphone (below 768px) */
@media (max-width: 767px) {
  body { font-size: 14px; }
  .sidebar { display: none; } /* Hide sidebar on mobile */
}
