Computer Keyboard

The keyboard is one of the most essential input devices in web development. Understanding its layout, key types, and shortcuts helps you code faster and more efficiently.

What is a Computer Keyboard?

A computer keyboard is an input device that allows users to enter data and commands into a computer by pressing keys. It is modelled after the typewriter keyboard and is the primary way we type code and text.

Keyboards communicate with the computer by sending electronic signals when each key is pressed. Modern keyboards connect via USB or Bluetooth.

Types of Keys

Key Type Examples Function
Alphanumeric Keys A–Z, 0–9 Used to type letters, numbers, and symbols
Function Keys F1 – F12 Perform special tasks depending on the program
Modifier Keys Shift, Ctrl, Alt Change the behaviour of other keys when held down
Navigation Keys Arrow keys, Home, End, Page Up/Down Move the cursor or scroll through documents
Special Keys Enter, Backspace, Delete, Tab, Esc Perform specific actions like confirming input or deleting text
Numeric Keypad 0–9, +, −, *, /, Enter Quick number entry, found on full-size keyboards
Symbol Keys !, @, #, $, %, &, <, > Used in programming for operators and syntax

Keys Used Frequently in HTML Coding

When writing HTML and CSS code, certain keys are used constantly. Below are the most important ones:

< (Less than) > (Greater than) / (Forward slash) = (Equals) " " (Double quotes) { } (Curly braces) : (Colon) ; (Semicolon) . (Period/Full stop) # (Hash) ! (Exclamation mark) ( ) (Parentheses) [ ] (Square brackets)
💡 Tip

The < and > keys (Shift + , and Shift + .) are used to open and close every HTML tag. Make sure you know their exact location on your keyboard!
