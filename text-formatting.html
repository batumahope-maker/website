Topic 7

Text Formatting Tags

HTML provides many tags for formatting text — making it bold, italic, underlined, highlighted, superscript, subscript, and more. These tags control both appearance and semantic meaning.

Heading Tags <h1> to <h6>

HTML provides 6 levels of headings. <h1> is the largest and most important; <h6> is the smallest. There should only be ONE <h1> per page.

HTML — Heading Tags
<h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>
Output

Heading Level 1

Heading Level 2

Heading Level 3

Heading Level 4

Heading Level 5
Heading Level 6

Live Code Examples

Bold and Italic

HTML
<p>This is <b>bold</b> and this is <strong>strongly important</strong>.</p>
<p>This is <i>italic</i> and this is <em>emphasised</em>.</p>
Output

This is bold and this is strongly important.

This is italic and this is emphasised.

Mark, Del, Ins, and Small

HTML
<p>The price was <del>$50</del> now <ins>$30</ins>.</p>
<p>Remember to <mark>study HTML</mark> for the exam.</p>
<p>Normal text. <small>This is smaller text.</small></p>
Output

The price was $50 now $30.

Remember to study HTML for the exam.

Normal text. This is smaller text.

Subscript and Superscript

HTML
<p>Water formula: H<sub>2</sub>O</p>
<p>Einstein: E = mc<sup>2</sup></p>
<p>CO<sub>2</sub> is carbon dioxide</p>
Output

Water formula: H2O

Einstein: E = mc2

CO2 is carbon dioxide

Blockquote and Inline Quote

HTML
<blockquote>
  "The only way to learn a new programming language
   is by writing programs in it." — Dennis Ritchie
</blockquote>

<p>As Einstein said: <q>Imagination is more important than knowledge.</q></p>
Output
"The only way to learn a new programming language is by writing programs in it." — Dennis Ritchie

As Einstein said: Imagination is more important than knowledge.

Code Tag

HTML
<p>Use the <code>print()</code> function to display output in Python.</p>
<p>In CSS, use <code>color: red;</code> to change text colour.</p>
Output

Use the print() function to display output in Python.

In CSS, use color: red; to change text colour.

Paragraph and Line Break Tags

HTML
<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>

<!-- Line break inside a paragraph -->
<p>Line one.<br>Line two on a new line.<br>Line three.</p>

<!-- Horizontal rule -->
<hr>
Output

This is the first paragraph.

This is the second paragraph.

Line one.
Line two on a new line.
Line three.
