Table Tags

HTML tables allow you to display data in a structured, grid-like format of rows and columns. They are used for schedules, databases, and any tabular data.

What is an HTML Table?

An HTML table is used to display data in a structured format consisting of rows and columns. Tables are created with the <table> element and a set of related tags.

Table Elements Explained

Tag Full Name Description
<table>TableDefines and wraps the entire table structure
<tr>Table RowDefines a horizontal row in the table
<th>Table HeaderDefines a header cell — bold and centred by default
<td>Table DataDefines a regular data cell containing content
<caption>CaptionOptional — provides a title/description above the table
<thead>Table HeadGroups the header rows (optional but good practice)
<tbody>Table BodyGroups the main body rows
<tfoot>Table FooterGroups the footer rows

Basic Table Syntax

HTML — Basic Table
<table border="1">
  <caption>Student Scores</caption>
  <tr>
    <th>Name</th>
    <th>Subject</th>
    <th>Score</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>Web Development</td>
    <td>95</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>Web Development</td>
    <td>88</td>
  </tr>
</table>
Output
Student Scores
Name Subject Score
Alice Web Development 95
Bob Web Development 88

Table Attributes

colspan — Span Multiple Columns

Used with <td> or <th> to make a cell span across multiple columns.

HTML
<td colspan="2">This cell spans two columns</td>

rowspan — Span Multiple Rows

Used with <td> or <th> to make a cell span across multiple rows.

HTML
<td rowspan="2">This cell spans two rows</td>

colspan + rowspan Example

HTML — colspan & rowspan Demo
<table border="1">
  <tr>
    <th colspan="3">School Timetable</th>
  </tr>
  <tr>
    <td rowspan="2">Monday</td>
    <td>HTML</td>
    <td>09:00</td>
  </tr>
  <tr>
    <td>CSS</td>
    <td>11:00</td>
  </tr>
</table>
Output
School Timetable
Monday HTML 09:00
CSS 11:00

Full Table with thead, tbody, tfoot

HTML — Structured Table
<table border="1">
  <thead>
    <tr>
      <th>Item</th>
      <th>Qty</th>
      <th>Price</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Notebook</td>
      <td>2</td>
      <td>$5.00</td>
    </tr>
    <tr>
      <td>Pen</td>
      <td>5</td>
      <td>$1.50</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">Total</td>
      <td>$17.50</td>
    </tr>
  </tfoot>
</table>
Output
Item Qty Price
Notebook2$5.00
Pen5$1.50
Total $17.50
📌 Note

The border attribute in HTML5 is deprecated. Use CSS border properties to style table borders instead.
