Topic 5

Media Tags

HTML media tags allow you to embed images, audio, and video directly into your web pages without needing external players or plugins.

Overview of Media Tags

TagMedia TypeDescription
<img>ImageEmbeds an image file (JPG, PNG, GIF, SVG, WebP)
<audio>AudioEmbeds an audio file (MP3, OGG, WAV)
<video>VideoEmbeds a video file (MP4, OGG, WebM)
<source>SourceSpecifies alternative media sources for <audio> or <video>
<iframe>ExternalEmbeds external content like YouTube videos or maps
<embed>External fileEmbeds a PDF, Flash animation or other file
<canvas>DrawingCreates a drawing surface using JavaScript
<svg>Vector graphicDefines scalable vector graphics in XML

1. The <img> Tag

The <img> tag is an unpaired (self-closing) tag used to display images. It has no closing tag.

Important Attributes

AttributeDescription
srcThe path or URL of the image file (required)
altAlternative text if image fails to load (required for accessibility)
widthWidth of the image in pixels
heightHeight of the image in pixels
HTML — Image Tag
<!-- Basic image -->
<img src="photo.jpg" alt="A scenic landscape">

<!-- Image with dimensions -->
<img src="logo.png" alt="Company Logo" width="200" height="100">

<!-- Image from the web -->
<img src="https://example.com/image.jpg" alt="Example image">
Output — Image Placeholder Example
[Image: photo.jpg]
120×80px

If the image cannot load, the alt text is displayed instead. This also helps screen readers describe the image to visually impaired users.

2. The <audio> Tag

The <audio> tag embeds audio files. Use the controls attribute to show play/pause buttons.

HTML — Audio Tag
<audio controls>
  <source src="music.mp3" type="audio/mp3">
  <source src="music.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Audio Attributes

AttributeEffect
controlsShows play, pause, and volume controls
autoplayStarts playing automatically when the page loads
loopRepeats the audio when it ends
mutedStarts the audio muted

3. The <video> Tag

The <video> tag embeds a video player. Always include width and height to prevent the page flickering while it loads.

HTML — Video Tag
<video width="640" height="360" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

<!-- Autoplay (muted required in modern browsers) -->
<video width="320" height="240" autoplay muted loop>
  <source src="background.mp4" type="video/mp4">
</video>
📌 Note

The <source> tag lets you provide multiple file formats. The browser picks the first one it supports. This ensures compatibility across Chrome, Firefox, Safari and other browsers.

4. The <iframe> Tag

The <iframe> (Inline Frame) tag embeds external content such as a YouTube video or Google Map directly inside your page.

HTML — Embedding a YouTube Video
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  frameborder="0"
  allowfullscreen>
</iframe>

5. SVG vs Canvas

FeatureSVGCanvas
TypeVector (XML-based)Raster (pixel-based)
ScalabilityScales without quality lossMay look blurry when scaled
AccessibilityElements are in the DOMNot accessible by default
ScriptingStyled via CSSDrawn via JavaScript
Use CaseIcons, diagrams, logosGames, dynamic drawing
HTML — SVG Circle Example
<svg width="200" height="100">
  <circle cx="50" cy="50" r="40" fill="#00e5c0"></circle>
  <rect x="110" y="10" width="80" height="80" fill="#ff6b6b"></rect>
</svg>
